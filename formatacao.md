## Negrito, Itálico e Riscado 

```markdown
**Texto em negrito**

*texto em ítalico*

~~Texto riscado~~

```
### Resultado:

**Texto em Negrito**

*Texto em ítalico*

~~Texto riscado~~

## Combinações de Formatação

```markdown
**texto em negrito e *ítalico***
```

## Código Inline

### Java Script
```javascript
const mensagem = "Olá, Mundo!";
console.log ('mensagem: ${Mesagem}');
```
### GO
```go

import "fmt"

func main() {
    mensagem := "Olá, Mundo!"
    fmt.Prinln ("Mensagem:"), mensagem
}
```
